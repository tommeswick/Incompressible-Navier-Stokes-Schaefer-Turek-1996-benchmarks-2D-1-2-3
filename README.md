In this repo, we consider the incompressible Navier-Stokes equations. Therein, the Schaefer-Turek-1996 benchmarks 2D-1, 2D-2, 2D-3 are computed; see here the paper https://wwwold.mathematik.tu-dortmund.de/lsiii/cms/papers/SchaeferTurek1996.pdf . The code is based on the deal.II library www.dealii.org version 9.6.0. If you have questions, don't hesitate to contact Thomas Wick, thomas.wick@ifam.uni-hannover.de
